# mobiuslab

A React + TypeScript web application with authentication and file management.

## Features

- **Authentication**: Login system with persistent sessions using localStorage
- **Dashboard**: Main workspace landing page
- **Downloads**: File management page with download links
- **Navigation**: Seamless routing between pages with logout functionality

## Tech Stack

- React 18
- TypeScript
- React Router
- Dracula Dark theme styling

## Pages

- `/login` - Authentication page
- `/dashboard` - Main workspace
- `/downloads` - File downloads section

## Styling

Custom Dracula Dark theme with purple accents:
- Background: `#282a36`
- Cards: `#44475a`
- Primary: `#bd93f9`
- Accent: `#ff79c6`
- Text: `#f8f8f2`
